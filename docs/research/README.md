# Research Archive

Early-stage research from project inception (2026-03). These files informed
the design tenets in DESIGN.md and specific feature decisions, but are not
actively maintained.

For current design decisions, see `docs/decisions/` (ADR format).
For feature specs, see `docs/specs/`.

## Files

- `config-system-analysis.md`: config architecture options (informed Phase 4a spec)
- `autohide-panels-design.md`: panel auto-hide research (informed Phase 1c)
- `swiftui-primitives.md`: SwiftUI layout patterns for terminal chrome
- `terminal-ux-patterns.md`: terminal UX survey across Ghostty, kitty, WezTerm, iTerm2
- `terminal-feature-requests.md`: feature request analysis from Ghostty discussions
- `cmux-patterns.md`: multiplexer pattern analysis (tmux, zellij, screen)
- `ux-brainstorm.md`: initial UX brainstorming session
- `feature-debate.md`: feature prioritization debate
- `improvement-plan.md`: early improvement roadmap (superseded by ROADMAP.md)
- `plan-review.md`: review of the initial implementation plan
